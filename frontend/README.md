# ASHEL_App

Smart application for Tunisian citizens to manage fines, access services, and simplify daily tasks.

##  Technologies
- Frontend: React (Vite)
- Backend: Java (Spring Boot)
- Database: MySQL / Oracle
- Other: Python

##  Features
- User authentication (CIN login)
- E-amende management
- E-administration services
- Complaint system
